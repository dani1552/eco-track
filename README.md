<h3>🌎 ECO TRACK - 지속 가능한 미래를 위한 나만의 탄소 발자국 감소 가이드</h1>
<br>
<hr />
<br>

### **👋🏻 서비스 목적**

- 기후 변화가 심각한 문제로 대두되고 있지만, 많은 사람들은 일상생활에서 탄소 발자국을 줄이는 방법을 찾고 꾸준히 실천하는 데 어려움을 겪음<br>
- 개인별 탄소 발자국 사용량을 측정하고 목표 점수를 설정한 뒤 구체적인 실천 방안을 제안하여 지속 가능한 생활 습관을 형성하도록 도움<br>

<br>

### **🏃🏻 타겟 사용자**

- 환경 보호에 관심은 있지만, 일상에서 어떤 실천을 해야 할지 구체적인 가이드가 없어 어려움을 느끼는 사용자
- 기후 변화의 심각성을 인지하고 있지만, 혼자 실천하기엔 지속적인 동기부여가 부족해 실천을 망설이는 사용자
